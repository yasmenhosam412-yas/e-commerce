# 📱 Flutter E-Commerce App

A modern **Flutter E-Commerce application** that allows users to browse products, explore collections, and place orders easily.  
The app provides a smooth shopping experience with a clean UI and scalable architecture.

---

## ✨ Features

- 🛍 Browse products by categories
- 🔍 Product details with images and prices
- ⭐ Product reviews and ratings
- 🛒 Shopping cart functionality
- 📦 Order management
- 🔐 Authentication
- 🌍 Multi-language support

---

## 🛠 Tech Stack

- **Flutter**
- **Firebase**
- **Cubit State Management**
- **Localization (ARB)**
